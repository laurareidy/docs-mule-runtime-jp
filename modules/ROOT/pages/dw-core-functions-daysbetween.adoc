[[daysbetween1]]
== daysBetween&#40;Date, Date&#41;: Number

2 つの日付間の日数を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `from` | 開始日 (`Date` (日付) 型)。
| `to` | 終了日 (`Date` (日付) 型)。
|===

=== 例

次の例では、指定された日付間の日数を返します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "days" : daysBetween("2016-10-01T23:57:59-03:00",
  "2017-10-01T23:57:59-03:00")
}
----

==== 出力

[source,JSON,linenums]
----
 {
   "days" : 365
 }
----
