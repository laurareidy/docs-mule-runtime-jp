[[unzip1]]
== unzip&#40;Array<Array<T&#62;&#62;&#41;: Array<Array<T&#62;&#62;

`zip` の逆の操作を実行します。


入力が 1 つの配列で、各インデックスに要素が 2 つの配列が含まれる場合、`unzip` の出力は 2 つの別個の配列になり、それぞれに各ペアの対応する要素が含まれます。入力配列のインデックスに 2 つ以上の要素の配列が含まれる場合、出力には、各インデックスの要素と同じ数の配列が含まれます。

=== 例

次の例では、配列の配列を展開します。例 `b` では、入力配列内の要素数が一定ではありません。そのため、関数は可能な限り多くの完全な配列を作成します。この場合は、1 つのみとなります。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "a" : unzip([[0,"a"],[1,"b"],[2,"c"],[3,"d"]]),
  "b" : unzip([ [0,"a"], [1,"a","foo"], [2], [3,"a"]])
}
----

==== 出力

[source,JSON,linenums]
----
{
   "a":[
      [0, 1, 2, 3],
      ["a", "b", "c", "d"]
    ],
  "b": [
      [0,1,2,3]
    ]
}
----
