[[ceil1]]
== ceil&#40;Number&#41;: Number

入力数値を最も近い整数に丸めます。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | 丸める数値。
|===

=== 例

次の例では、数値を最も近い整数に丸めます。`2.1` は 
`3` に丸められています。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---

{
  a: ceil(1.5),
  b: ceil(2.1),
  c: ceil(3)
}
----

==== 出力

[source,JSON,linenums]
----
{
  "a": 2,
  "b": 3,
  "c": 3
}
----
