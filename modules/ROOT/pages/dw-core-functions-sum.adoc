[[sum1]]
== sum&#40;Array<Number&#62;&#41;: Number

配列内の数値の合計を返します。


配列が空の場合は `0` を返し、配列内に数値以外の値がある場合はエラーになります。

=== 例

次の例では、入力配列内の値の合計を返します。

==== ソース
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
sum([1, 2, 3])
----

==== 出力
[source,JSON,linenums]
----
6
----
