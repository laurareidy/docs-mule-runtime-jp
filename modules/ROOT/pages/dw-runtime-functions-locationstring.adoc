[[locationstring1]]
== locationString&#40;Any&#41;: String

指定された値の位置の文字列を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前) | 説明
| `value` | 任意の型の値。
|===

=== 例

次の例では、DataWeave スクリプトのヘッダー内で変数 `a` を定義する行 (位置) のコンテンツを返します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
import * from dw::Runtime
output application/json
var a = 123
---
locationString(a)
----

==== 出力

[source,JSON,linenums]
----
"var a = 123"
----
