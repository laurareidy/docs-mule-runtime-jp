このモジュールには、文字列を操作するヘルパー関数が含まれます。

このモジュールを使用するには、DataWeave コードにインポートする必要があります。その方法として、たとえば、行 `import * from dw::core::URL` を DataWeave スクリプトの
ヘッダーに追加します。
