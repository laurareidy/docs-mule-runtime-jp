[[iseven1]]
== isEven&#40;Number&#41;: Boolean

特定の数値が偶数の場合は `true`、それ以外の場合は `false` を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | 評価する数値。
|===

=== 例

次の例では、入力数値が偶数かどうかを示します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output  application/json
---
{
   "isEven" :
   [ isEven(0), isEven(1), isEven(2)]
}
----

==== 出力

[source,JSON,linenums]
----
{
  "isEven" : [
    true,
    false,
    true
  ]
}
----
