[[min1]]
== min&#40;Array<T&#62;&#41;: T &#124; Null

配列内の最小数値を返します。


配列が空の場合は null を返し、配列に数値以外の値がある場合はエラーになります。

=== 例

次の例では、各入力配列の最小数値を返します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: min([1, 1000]),
  b: min([1, 2, 3]),
  d: min([1.5, 2.5, 3.5])
}
----

==== 出力

[source,JSON,linenums]
----
{
  "a": 1,
  "b": 1,
  "d": 1.5
}
----
