[[isinteger1]]
== isInteger&#40;Number&#41;: Boolean

特定の数値が整数 (小数部なし) の場合は `true`、それ以外の場合は `false` を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | 評価する数値。
|===

=== 例

次の例では、入力が整数かどうかを示します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
{
   "integer" : isInteger(1.1),
   "integer" : isInteger(1)
}
----

==== 出力

[source,JSON,linenums]
----
{
   "integer": false,
   "integer": true
}
----
