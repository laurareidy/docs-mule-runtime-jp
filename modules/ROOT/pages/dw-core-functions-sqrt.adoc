[[sqrt1]]
== sqrt&#40;Number&#41;: Number

入力数値の平方根を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | この操作を適用する数値。
|===

=== 例

次の例では、入力数値の平方根を返します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "a" : sqrt(4),
  "b" : sqrt(25),
  "c" : sqrt(100)
}
----

==== 出力

[source,JSON,linenums]
----
{
  "a": 2.0,
  "b": 5.0,
  "c": 10.0
}
----
