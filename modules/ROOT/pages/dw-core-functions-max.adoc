[[max1]]
== max&#40;Array<T&#62;&#41;: T &#124; Null

リスト (配列) 内の最大数値を返します。


配列が空の場合は null を返し、配列に数値以外の値がある場合はエラーになります。

=== 例

次の例では、各入力配列の最大値を返します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: max([1, 1000]),
  b: max([1, 2, 3]),
  d: max([1.5, 2.5, 3.5])
}
----

==== 出力

[source,JSON,linenums]
----
{
  "a": 1000,
  "b": 3,
  "d": 3.5
}
----
