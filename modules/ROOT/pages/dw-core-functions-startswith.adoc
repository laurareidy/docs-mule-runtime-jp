== startsWith&#40;String, String&#41;: Boolean

入力文字列が照合するプレフィックスで始まるかどうかによって `true` または `false` を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `text` | 入力文字列。
| `prefix` | プレフィックスを識別する文字列。
|===

=== 例

次の例では、文字列が指定したプレフィックスで始まるかどうかを示します。`startsWith(text,prefix)` 表記 (`startsWith("Mariano","Mar")` など) を使用することも
できます。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "yes" : "Mariano" startsWith "Mar",
  "no" : "Mariano" startsWith "Em"
}
----

==== 出力

[source,JSON,linenums]
----
{
  "yes": true,
  "no": false
}
----
