Remove Variable コンポーネントは、変数名を受け入れて、Mule イベントから変数を削除します。

[%header,cols="30a,70a"]
|===
|項目 | 説明

| Display Name (表示名) (`doc:name`)
| このコンポーネントの一意の名前をフローで表示するようにカスタマイズします。デフォルトは Remove Variable です。

| Name (名前) (`variableName`)
| 削除する変数の名前。
|===
