[[wait1]]
== wait&#40;T, Number&#41;: T

指定されたタイムアウト (ミリ秒) の間、実行を停止します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前) | 説明
| `value` | 任意の型の入力。
| `timeout` | 待機する時間 (ミリ秒)。
|===

=== 例

次の例では、2000 ミリ秒 (2 秒) の間、実行を待機します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
import * from dw::Runtime
output application/json
---
{ "user" : 1 } wait 2000
----

==== 出力

[source,JSON,linenums]
----
{ "user": 1 }
----
