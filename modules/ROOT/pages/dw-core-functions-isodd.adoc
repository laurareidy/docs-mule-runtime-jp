[[isodd1]]
== isOdd&#40;Number&#41;: Boolean

特定の数値が奇数の場合は `true`、それ以外の場合は `false` を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | 評価する数値 (`Number` (数値) 型)。
|===

=== 例

次の例では、入力数値が奇数かどうかを示します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output  application/json
---
{
   "isOdd" :
   [ isOdd(0), isOdd(1), isOdd(2) ]
}
----

==== 出力

[source,JSON,linenums]
----
{
  "isOdd": [
    false,
    true,
    false
  ]
}
----
