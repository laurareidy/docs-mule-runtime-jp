[[randomint1]]
== randomInt&#40;Number&#41;: Number

0 以上指定数値未満の整数の疑似乱数を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `upperBound` | 乱数の上限を設定する数値。
|===

=== 例

次の例では、0 以上 1000 未満の整数を返します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
{ price: randomInt(1000) }
----

==== 出力

[source,JSON,linenums]
----
{ "price": 442.0 }
----
