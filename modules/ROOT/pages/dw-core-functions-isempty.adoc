[[isempty1]]
== isEmpty&#40;Array<Any&#62;&#41;: Boolean

特定のリスト (配列) が空の場合は `true`、それ以外の場合は `false` を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `elements` | 評価するリスト (配列)。
|===

=== 例

次の例では、入力配列が空かどうかを示します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
{
  "empty" : isEmpty([]),
  "nonEmpty" : isEmpty([1])
}
----

==== 出力

[source,JSON,linenums]
----
{
   "empty": true,
   "nonEmpty": false
}
----
