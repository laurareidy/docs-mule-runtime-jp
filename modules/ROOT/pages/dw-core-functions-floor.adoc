[[floor1]]
== floor&#40;Number&#41;: Number

入力数値の小数点以下を切り捨てて最も近い整数にします。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | この操作を適用する数値。
|===

=== 例

次の例では、数値の小数点以下を切り捨てて最も近い整数にします。`1.5` の小数点以下が
切り捨てられて `1` になっています。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "a" : floor(1.5),
  "b" : floor(2.2),
  "c" : floor(3)
}
----

==== 出力

[source,JSON,linenums]
----
{
  "a": 1,
  "b": 2,
  "c": 3
}
----
