[[isdecimal1]]
== isDecimal&#40;Number&#41;: Boolean

特定の数値に小数部が含まれる場合は `true`、それ以外の場合は `false` を返します。

=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | 評価する数値。
|===

=== 例

次の例では、入力数値に小数部が含まれるかどうかを示します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output application/json
---
{
  "decimal" : isDecimal(1.1),
  "decimal" : isDecimal(1)
}
----

==== 出力

[source,JSON,linenums]
----
  {
    "decimal": true,
    "decimal": false
  }
----
