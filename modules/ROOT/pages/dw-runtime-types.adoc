a| TryResult
a| `type TryResult = { success: Boolean, result?: T, error?: { kind: String, message: String, stack?: Array<String&#62;, location?: String } }`
a| オブジェクトと結果またはエラーメッセージ。`success` が `false` の場合は、`error` が含まれます。`true` の場合は `result` が含まれます。
