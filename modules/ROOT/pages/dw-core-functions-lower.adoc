[[lower1]]
== lower&#40;String&#41;: String

指定された文字列を小文字で返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `text` | 文字列 (`String` (文字列) 型)。
|===

=== 例

次の例では、大文字を小文字に変換します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "name" : lower("MULESOFT")
}
----

==== 出力

[source,JSON,linenums]
----
{
  "name": "mulesoft"
}
----

== lower&#40;Null&#41;: Null
