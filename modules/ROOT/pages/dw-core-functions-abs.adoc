[[abs1]]
== abs&#40;Number&#41;: Number

入力数値の絶対値を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `number` | この操作を適用する数値。
|===

=== 例

次の例では、入力数値の絶対値を返します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: abs(-2),
  b: abs(2.5),
  c: abs(-3.4),
  d: abs(3)
}
----

==== 出力

[source,JSON,linenums]
----
{
  "a": 2,
  "b": 2.5,
  "c": 3.4,
  "d": 3
}
----
