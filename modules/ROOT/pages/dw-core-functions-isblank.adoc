[[isblank1]]
== isBlank&#40;String &#124; Null&#41;: Boolean

特定の文字列が空か、完全な空白の場合は `true`、
それ以外の場合は `false` を返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前)   | 説明
| `text` | 評価する文字列。
|===

=== 例

次の例では、特定の値が空白かどうかを示します。

==== ソース

[source,DataWeave,linenums]
----
%dw 2.0
output  application/json
---
{
  "empty" : isBlank(""),
  "withSpaces" : isBlank("      "),
  "withText" : isBlank(" 1223")
}
----

==== 出力

[source,JSON,linenums]
----
{
  "empty": true,
  "withSpaces": true,
  "withText": false
}
----
