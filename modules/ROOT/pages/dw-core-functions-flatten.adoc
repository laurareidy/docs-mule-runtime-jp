[[flatten1]]
== flatten&#40;Array<Array<T&#62; &#124; Q&#62;&#41;: Array<T &#124; Q&#62;

配列の配列を 1 つの単純な配列にフラット化します。


=== 例

次の例では、配列の配列をフラット化します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
flatten(payload)
----

==== 入力

[source,JSON,linenums]
----
[
   [3,5],
   [9,5],
   [154,0.3]
]
----

==== 出力

[source,JSON,linenums]
----
[
  3,
  5,
  9,
  5,
  154,
  0.3
]
----

== flatten&#40;Null&#41;: Null
