[[valueset1]]
== valueSet&#40;{ &#40;K&#41;?: V }&#41;: Array<V&#62;

オブジェクト内の値のリストを返します。


=== パラメータ

[%header, cols="1,3"]
|===
| Name (名前) | 説明
| `obj` | 評価するオブジェクト。
|===

=== 例

次の例では、入力オブジェクトからの値を返します。

==== ソース

[source,DataWeave, linenums]
----
%dw 2.0
import * from dw::core::Objects
output application/json
---
{ "valueSet" : valueSet({a: true, b: 1}) }
----

==== 出力

[source,JSON,linenums]
----
{ "valueSet" : [true,1] }
----
